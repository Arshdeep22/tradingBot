## 1. The Foundation: Why Zones Work

### The Institutional Order Theory

Professional zone traders believe:

1. **Large institutions** (banks, mutual funds, pension funds) trade in massive quantities — millions of shares
2. They **cannot execute** their full order at one price without moving the market against themselves
3. So they execute in **tranches** — filling some orders now, leaving the rest as pending orders
4. The area where they first executed creates a **zone** (supply or demand)
5. When price returns to that zone, the **remaining unfilled orders activate**, causing price to move again
6. This is why **fresh zones work** — unfilled orders still exist there

### Why First Touch is Best

```
Zone formed → Institution fills 60% of order
Price leaves → Market moves away
Price returns (1st time) → Remaining 40% activates → Strong reaction
Price returns (2nd time) → Maybe 10% left → Weak reaction
Price returns (3rd time) → No orders left → Zone breaks
```

**Rule: Only trade the FIRST return to a fresh zone.**

### Supply vs Demand

| | Demand Zone | Supply Zone |
|---|---|---|
| **What happened** | Institutions BOUGHT heavily | Institutions SOLD heavily |
| **Price action** | Price dropped, paused, then rallied sharply | Price rallied, paused, then dropped sharply |
| **Unfilled orders** | Buy orders remain at that level | Sell orders remain at that level |
| **Trade** | BUY when price returns | SELL when price returns |
| **Psychology** | "Buyers overwhelmed sellers here" | "Sellers overwhelmed buyers here" |

## 2. Zone Formation Patterns

### The 4 Core Patterns

Professionals identify exactly **4 patterns** for zone formation:

### Pattern 1: DROP-BASE-RALLY (DBR) — Demand Zone

```
    │
    │  ← LEG IN (Drop)
    │
    ├──────┤  ← BASE (1-3 small candles)
    │
    │      │
    │      │  ← LEG OUT (Rally)
    │      │
           │
```

- **What it means**: Sellers were in control (drop), institutions stepped in (base), buyers took over (rally)
- **Zone**: The BASE area — where institutions placed their buy orders
- **Trade**: BUY when price returns to this base

### Pattern 2: RALLY-BASE-DROP (RBD) — Supply Zone

```
           │
    │      │  ← LEG IN (Rally)
    │      │
    │
    ├──────┤  ← BASE (1-3 small candles)
    │
    │  ← LEG OUT (Drop)
    │
```

- **What it means**: Buyers were in control (rally), institutions stepped in (base), sellers took over (drop)
- **Zone**: The BASE area — where institutions placed their sell orders
- **Trade**: SELL when price returns to this base

### Pattern 3: RALLY-BASE-RALLY (RBR) — Continuation Demand

```
           │
    │      │  ← LEG OUT (Rally continues)
    │      │
    │
    ├──────┤  ← BASE (pause/consolidation)
    │
    │      │
    │      │  ← LEG IN (Rally)
    │      │
```

- **What it means**: Buyers in control (rally), brief pause (base), then continued buying (rally)
- **Zone**: The BASE area — institutions added MORE buy orders during the pause
- **Trade**: BUY when price returns (buying the pullback in an uptrend)
- **Why it works**: In an uptrend, institutions keep adding to positions on pullbacks

### Pattern 4: DROP-BASE-DROP (DBD) — Continuation Supply

```
    │
    │  ← LEG IN (Drop)
    │
    ├──────┤  ← BASE (pause/consolidation)
    │
    │  ← LEG OUT (Drop continues)
    │
```

- **What it means**: Sellers in control (drop), brief pause (base), then continued selling (drop)
- **Zone**: The BASE area — institutions added MORE sell orders during the pause
- **Trade**: SELL when price returns (selling the rally in a downtrend)

### Which Patterns Are Strongest?

| Pattern | Strength | Reason |
|---------|----------|--------|
| RBD (Supply) | ★★★★★ | Reversal from up to down — maximum surprise |
| DBR (Demand) | ★★★★★ | Reversal from down to up — maximum surprise |
| DBD (Supply) | ★★★★☆ | Continuation — trend adds conviction |
| RBR (Demand) | ★★★★☆ | Continuation — trend adds conviction |

**All 4 are tradeable. Reversal patterns have slightly higher reward potential, continuation patterns have higher probability.**

## 3. How to Mark Zones Like a Professional

### Step-by-Step Zone Marking Process

#### Step 1: Identify the Explosive Move (Leg-Out)

The leg-out is **the most important candle**. Look for:

| Criteria | What to Look For | Professional Standard |
|----------|-----------------|---------------------|
| **Size** | Candle body much larger than average | Body ≥ 2x average body size |
| **Body ratio** | Body takes up most of the candle (small wicks) | Body ≥ 60% of total High-Low range |
| **Color** | Clear direction (bullish for demand, bearish for supply) | Must be in the correct direction |
| **Consecutive** | Multiple large candles = even stronger | 2-3 large candles = very strong zone |
| **Gap** | Price gap away from zone | Gap = strongest possible signal |
| **Volume** | High volume on the leg-out | Volume > 1.5x average |

**Leg-out is what CONFIRMS a zone exists. Without a strong leg-out, there is no zone.**

#### Step 2: Find the Base (The Actual Zone)

Go BACK from the leg-out to find the consolidation:

| Criteria | Professional Standard |
|----------|---------------------|
| **Candle count** | 1-3 candles (MAXIMUM 5, but prefer 1-2) |
| **Candle type** | Doji, spinning top, inside bar, small body candles |
| **Candle size** | Body < 50% of average body size |
| **Range** | Candles overlap each other (consolidation) |
| **Wicks** | Wicks show rejection (hammer, inverted hammer) |

**The tighter the base, the stronger the zone. 1 candle base = "Drop-Base-Rally" at its finest.**

#### Step 3: Draw the Zone Boundaries

```
ZONE TOP    = Highest HIGH of ALL base candles (include wicks!)
ZONE BOTTOM = Lowest LOW of ALL base candles (include wicks!)
```

**Important**: Include the WICKS, not just bodies. The entire price range of the base candles is the zone.

#### Visual Example:

```
                    Zone Marking Example (Demand Zone)
                    
Price
 │
 │              ┌─┐
 │         ┌─┐ │ │   ← LEG OUT (large bullish candles)
 │         │ │ │ │
 │    ─────┤ ├─┤ ├─────── Zone Top (highest high of base)
 │    │ ┌┐ │     │
 │    │ ├┤ │     │   ← BASE (2 small candles in between)
 │    └─┤├─┘     │
 │    ─────────────────── Zone Bottom (lowest low of base)
 │  ┌─┐
 │  │ │              ← LEG IN (bearish candle dropping into base)
 │  │ │
 │  └─┘
 └─────────────────────── Time
```

#### Step 4: Validate the Leg-In

The leg-in confirms the imbalance:

| For Demand Zone | For Supply Zone |
|----------------|-----------------|
| Leg-in must be BEARISH (dropping) | Leg-in must be BULLISH (rallying) |
| Should be a meaningful move (not tiny) | Should be a meaningful move (not tiny) |
| Body > 50% of average body OR multi-candle drop | Body > 50% of average body OR multi-candle rally |
| Shows that sellers were in control BEFORE institutions stepped in | Shows that buyers were in control BEFORE institutions stepped in |

**For continuation patterns (RBR/DBD):**
- RBR: Leg-in is BULLISH (rally → base → rally)
- DBD: Leg-in is BEARISH (drop → base → drop)

#### Step 5: Determine Zone Freshness

```
FRESH ZONE = Price has NEVER returned to the zone since formation

Price returning to zone means:
- For DEMAND: Price's LOW touches or enters zone_top → zone_bottom range
- For SUPPLY: Price's HIGH touches or enters zone_bottom → zone_top range

Even a WICK into the zone = zone is tested (not fresh)
```

**Critical Rule**: Only trade FRESH zones. Period.

| Zone Status | What Happened | Action |
|-------------|--------------|--------|
| **Fresh** | Price never returned since formation | ✅ TRADE IT |
| **Tested once** | Price returned once, reacted | ❌ Do NOT trade (some orders filled) |
| **Broken** | Price went through the zone | ❌ Zone is DEAD — remove it |

## 4. Zone Quality Assessment

### The Professional Scoring Framework

Professionals assess zone quality on these dimensions:

### Dimension 1: Departure Strength (How explosively price left)

| Level | Criteria | Score |
|-------|----------|-------|
| **EXCEPTIONAL** | Gap away OR 3+ consecutive large candles | 10/10 |
| **STRONG** | 2 large candles with body > 2x average | 8/10 |
| **GOOD** | 1 large candle with body > 2x average, full body (>70% of range) | 6/10 |
| **MODERATE** | 1 large candle with body > 1.5x average | 4/10 |
| **WEAK** | Small leg-out, wicky candle | 2/10 — Don't trade |

### Dimension 2: Base Quality (How tight is the consolidation)

| Level | Criteria | Score |
|-------|----------|-------|
| **EXCELLENT** | 1 candle base (instant reaction) | 10/10 |
| **VERY GOOD** | 2 candle base, both small | 8/10 |
| **GOOD** | 3 candle base, overlapping | 6/10 |
| **MODERATE** | 4-5 candle base | 4/10 |
| **POOR** | 6+ candle base (too much time spent) | 2/10 — Don't trade |

### Dimension 3: Freshness

| Level | Criteria | Score |
|-------|----------|-------|
| **PRISTINE** | Never tested, formed recently (within 50 candles) | 10/10 |
| **FRESH** | Never tested, formed 50-100 candles ago | 7/10 |
| **AGING** | Never tested, formed 100+ candles ago | 4/10 |
| **TESTED** | Price returned to zone once | 0/10 — Don't trade |

### Dimension 4: Arrival Strength (How price approaches the zone NOW)

This is what most amateurs miss. When price is RETURNING to the zone, HOW it arrives matters:

| Level | Criteria | Score |
|-------|----------|-------|
| **IDEAL** | Price approaching slowly, small candles, grinding toward zone | 10/10 |
| **GOOD** | Price approaching at moderate pace | 7/10 |
| **POOR** | Price CRASHING toward zone with huge candles | 3/10 — Higher chance of breaking through |

**Why arrival matters**: If price races toward a demand zone with massive bearish candles, the momentum may carry through the zone. Slow, grinding approach means sellers are exhausting — perfect for the demand zone to hold.

### Dimension 5: Time at Level (How long since zone formed)

| Level | Criteria | Score |
|-------|----------|-------|
| **RECENT** | Formed within last 1-2 days (intraday TF) | 10/10 |
| **MODERATE** | Formed 2-5 days ago | 7/10 |
| **OLD** | Formed 1-2 weeks ago | 4/10 |
| **ANCIENT** | Formed 2+ weeks ago | 2/10 — Probably no longer valid |

### Dimension 6: Trend Alignment

| Level | Criteria | Score |
|-------|----------|-------|
| **WITH TREND** | Demand zone in uptrend / Supply zone in downtrend | 10/10 |
| **NEUTRAL** | Zone in sideways market | 5/10 |
| **COUNTER-TREND** | Demand zone in downtrend / Supply zone in uptrend | 3/10 — Higher risk |

### Minimum Quality to Trade

**Professional standard**: Total score ≥ 40/60 (across 6 dimensions × 10 each)

Most professionals won't trade zones below 7/10 on Departure Strength and won't trade tested zones at all.

## 5. Entry Rules

### Entry Method 1: Limit Order at Zone Edge (Most Common)

```
DEMAND ZONE:
  Place BUY LIMIT ORDER at zone_top
  → You get filled when price drops to zone_top
  → This is the "first touch" entry

SUPPLY ZONE:
  Place SELL LIMIT ORDER at zone_bottom
  → You get filled when price rallies to zone_bottom
  → This is the "first touch" entry
```

**Pros**: Best possible entry price, tight stop loss  
**Cons**: No confirmation — price might break through

### Entry Method 2: Confirmation Candle (More Conservative)

```
DEMAND ZONE:
  1. Wait for price to ENTER the zone
  2. Wait for a BULLISH confirmation candle:
     - Bullish engulfing
     - Hammer/pin bar with long lower wick
     - Morning star
     - Any candle that closes ABOVE its midpoint after touching zone
  3. Enter on the CLOSE of confirmation candle (or next candle open)

SUPPLY ZONE:
  1. Wait for price to ENTER the zone
  2. Wait for a BEARISH confirmation candle:
     - Bearish engulfing
     - Shooting star/inverted hammer
     - Evening star
     - Any candle that closes BELOW its midpoint after touching zone
  3. Enter on the CLOSE of confirmation candle
```

**Pros**: Higher probability — you have visual proof of reaction  
**Cons**: Slightly worse entry price (you enter after the bounce starts)

### Entry Method 3: Zone Midpoint Entry (Aggressive)

```
DEMAND: Enter at (zone_top + zone_bottom) / 2
SUPPLY: Enter at (zone_top + zone_bottom) / 2
```

**Pros**: Better entry price, more room for SL  
**Cons**: Price may not reach midpoint — you might miss the trade

### Which Method Do Professionals Use?

| Trader Type | Method | Why |
|-------------|--------|-----|
| **Scalpers** | Limit at zone edge | Speed, tight risk |
| **Day traders** | Confirmation candle | Balance of probability and price |
| **Swing traders** | Limit at zone edge | Set and forget |
| **Algorithmic** | Limit at zone edge + auto-cancel if broken | Automated systems can't judge candle patterns easily |

**For our bot: Method 1 (Limit at zone edge) is most automatable. Method 2 (confirmation) could be added as an optional filter.**

## 6. Stop Loss Placement

### The Golden Rule

> **Stop loss goes where the zone idea is INVALIDATED — if price goes there, your thesis was wrong.**

### Method 1: Beyond Zone + ATR Buffer (Recommended for Automation)

```
DEMAND: SL = zone_bottom - (1.0 to 1.5 × ATR)
SUPPLY: SL = zone_top + (1.0 to 1.5 × ATR)
```

- ATR adapts to volatility
- 1.0x ATR = tighter (more gets stopped out, better R:R)
- 1.5x ATR = wider (fewer false stops, worse R:R)

### Method 2: Below/Above Leg-In Extreme

```
DEMAND: SL = Low of the leg-in candle - small buffer
SUPPLY: SL = High of the leg-in candle + small buffer
```

- Logic: If price goes below where the leg-in started, the entire pattern is broken
- Often gives a wider SL than Method 1

### Method 3: Fixed Percentage Below Zone

```
DEMAND: SL = zone_bottom × (1 - 0.002)  [0.2% below]
SUPPLY: SL = zone_top × (1 + 0.002)     [0.2% above]
```

- Simple but doesn't adapt to volatility
- Works OK for stocks with consistent volatility

### Professional SL Rules

| Rule | Details |
|------|---------|
| **Never move SL further away** | If you're tempted, your position is too large |
| **SL must be logical** | Beyond a level that, if broken, means your thesis is wrong |
| **Maximum SL = 2% of capital** | If zone requires > 2% risk, reduce position size or skip |
| **SL should be < 1.5% of price** | For intraday, SL beyond 1.5% of entry = too wide |
| **Account for spread/slippage** | Add 0.05-0.1% buffer for real-world execution |

### SL Placement Visual

```
DEMAND ZONE with SL:

    ENTRY ────────── 2450.00 (zone_top)
    ╔════════════╗
    ║   ZONE     ║   height = 8 pts (0.33%)
    ╚════════════╝
    Zone Bottom ──── 2442.00
    ↓ ATR Buffer ─── 12.50 (1.0 × ATR14)
    STOP LOSS ────── 2429.50
    
    Total Risk = 2450 - 2429.50 = 20.50 (0.84% of price) ✅
```

## 7. Target Setting

### Method 1: Fixed Risk:Reward Ratio

```
DEMAND: Target = Entry + (R:R × Risk)
SUPPLY: Target = Entry - (R:R × Risk)

Common ratios:
  1:2 → Conservative (higher win rate, lower reward)
  1:3 → Standard (balanced)
  1:4 → Aggressive (lower win rate, higher reward)
  1:5 → Very aggressive (use only for A+ setups)
```

### Method 2: Opposing Zone Target (Professional Preferred)

```
DEMAND ZONE target = Bottom of nearest SUPPLY zone above entry

Example:
  Demand zone entry at 2450
  Nearest supply zone above: 2520-2530
  Target = 2520 (bottom of supply zone)
  
  If R:R with this target is at least 1:2, TAKE THE TRADE
  If R:R is less than 1:2, SKIP (not enough reward)
```

**This is the most professional approach** — your target is at a level where you KNOW selling pressure exists.

### Method 3: Previous Structure (Swing Highs/Lows)

```
DEMAND: Target = Previous swing high
SUPPLY: Target = Previous swing low

Reasoning: Price often respects previous turning points
```

### Method 4: Partial Profits (The Professional Standard)

```
DEMAND ZONE trade at entry 2450, SL 2430, Risk = 20:

  Target 1: 2450 + (1 × 20) = 2470  → Take 50% profit, move SL to breakeven
  Target 2: 2450 + (2 × 20) = 2490  → Take 25% more, trail SL
  Target 3: 2450 + (3 × 20) = 2510  → Close remaining 25%

  OR

  Target 1: 2470 (1:1) → Take 50%, move SL to entry
  Target 2: Opposing zone → Close remaining 50%
```

### Target Decision Framework

| Situation | Best Target Method |
|-----------|-------------------|
| Clear opposing zone nearby | Target at opposing zone |
| No clear opposing zone | Fixed R:R (1:3) |
| Strong trend (continuation zone) | Trail stop, let it run |
| Choppy market | Take profit at 1:2, don't get greedy |
| A+ setup (score > 50/60) | Use aggressive target (1:4 or 1:5) |

## 8. Trade Management

### Once You're In a Trade

| Phase | Action |
|-------|--------|
| **Entry to 1:1 R:R** | Do NOTHING. Let it work. Don't micro-manage. |
| **At 1:1 R:R** | Option A: Take partial profit (50%), move SL to breakeven |
| | Option B: Move SL to breakeven only (no partial) |
| **1:1 to 1:2** | Trail stop below recent swing low (for longs) |
| **At target** | Close remaining position. Done. |
| **If SL hit** | Accept the loss. Move on. DO NOT re-enter same zone. |

### Professional Trade Management Rules

1. **Never add to a losing position** — If price is going against you, your analysis was wrong
2. **Never move SL further away** — This turns a defined risk into an undefined one
3. **Take 1-2 trades per day MAX** — Quality over quantity
4. **If 2 consecutive losses, STOP for the day** — Protect capital and psychology
5. **Journal every trade** — Zone screenshot, entry reason, outcome, what you'd do differently
6. **Don't chase** — If you missed the entry, wait for the NEXT zone
7. **Time-based exit** — If trade hasn't moved in your direction after 2x the number of base candles, consider closing at small loss/breakeven

### Trailing Stop Methods

```
Method 1: Swing Trail
  Move SL below each new higher low (for longs)
  Move SL above each new lower high (for shorts)

Method 2: ATR Trail
  SL = Current price - (2 × ATR) for longs
  SL = Current price + (2 × ATR) for shorts
  Update every candle close

Method 3: EMA Trail
  SL = EMA(9) for aggressive trail
  SL = EMA(21) for conservative trail
  Close when price closes below EMA
```

## 9. Multi-Timeframe Analysis

### The 3-Timeframe System (Professional Standard)

| Timeframe | Purpose | What You Do |
|-----------|---------|-------------|
| **Higher TF** (1H or Daily) | Determine TREND and find major zones | Only trade IN the direction of this TF's trend |
| **Trading TF** (15m) | Find YOUR zones | This is where you identify and score zones |
| **Entry TF** (5m or 3m) | Time your entry precisely | Refine entry, confirm reversal |

### How It Works in Practice

```
Step 1: DAILY/1H Chart
  - Is the overall trend UP, DOWN, or SIDEWAYS?
  - Are there any major supply/demand zones from this TF?
  - Where are the key support/resistance levels?
  
  → If UPTREND: Only look for DEMAND zones to buy
  → If DOWNTREND: Only look for SUPPLY zones to sell
  → If SIDEWAYS: Trade both, but with tighter targets

Step 2: 15m Chart
  - Find all fresh supply and demand zones
  - Score them using the 6-dimension framework
  - Identify which zones price is likely to reach today
  - Mark entry levels for the best zones (score ≥ 40/60)

Step 3: 5m/3m Chart (When price approaches your zone)
  - Watch HOW price approaches (arrival quality)
  - Look for confirmation candle (if using that entry method)
  - Set precise entry, SL, target levels
  - Execute the trade
```

### Multi-Timeframe Confluence

**The BEST trades happen when zones align across timeframes:**

```
Example:
  Daily: Major demand zone at 2440-2460 (formed 2 weeks ago, never tested)
  15m:   Fresh demand zone at 2445-2450 (formed yesterday)
  5m:    Price approaching zone slowly, small candles

  → This is an A+ setup. The 15m zone sits INSIDE the daily zone.
  → Maximum institutional support at this level.
  → Trade with full confidence.
```

### Timeframe Selection for Indian Markets (NSE)

| Trading Style | Higher TF | Trading TF | Entry TF |
|--------------|-----------|------------|----------|
| Scalping (5-15 min holds) | 15m | 5m | 1m |
| Intraday (30min-2hr holds) | 1H | 15m | 5m |
| Swing (1-5 day holds) | Daily | 1H | 15m |
| Positional (weeks) | Weekly | Daily | 1H |

**For our bot (intraday focus): 1H → 15m → 5m is the ideal combination.**

## 10. Risk Management & Position Sizing

### The 1% Rule

> **Never risk more than 1% of your total capital on a single trade.**

```
Capital = ₹1,00,000
Max risk per trade = 1% = ₹1,000

If entry = 2450 and SL = 2430:
  Risk per share = 2450 - 2430 = ₹20
  Position size = 1000 / 20 = 50 shares
  Total position value = 50 × 2450 = ₹1,22,500
```

### Position Sizing Formula

```
Position Size (shares) = Risk Amount / Risk Per Share

Where:
  Risk Amount = Capital × Risk Percentage (e.g., 1%)
  Risk Per Share = Entry Price - Stop Loss Price (for longs)
```

### Daily Risk Limits

| Rule | Value | Reason |
|------|-------|--------|
| Max risk per trade | 1% of capital | Survive losing streaks |
| Max daily loss | 2-3% of capital | Stop trading after 2-3 losses |
| Max open positions | 3-5 | Don't over-diversify |
| Max correlation | Don't hold same sector × 3 | Sector risk |
| Max consecutive losses before pause | 3 | Protect psychology |

### The Math of Zone Trading

With 1:3 R:R (risking 1 to make 3):

| Win Rate | Expected Value | Profitable? |
|----------|---------------|-------------|
| 25% | (0.25 × 3) - (0.75 × 1) = 0.00 | Breakeven |
| 30% | (0.30 × 3) - (0.70 × 1) = 0.20 | ✅ Profitable |
| 35% | (0.35 × 3) - (0.65 × 1) = 0.40 | ✅ Good |
| 40% | (0.40 × 3) - (0.60 × 1) = 0.60 | ✅ Very Good |
| 50% | (0.50 × 3) - (0.50 × 1) = 1.00 | ✅ Excellent |

**You only need 34% win rate to be profitable with 1:3 R:R!**  
Professional zone traders typically achieve 40-55% win rate.

| Period | Rule | Reason |
|--------|------|--------|
| 9:15 - 9:30 AM | ❌ NO TRADING | Opening volatility, gaps, news reaction |
| 9:30 - 9:45 AM | ⚠️ Be cautious | Still settling |
| 9:45 AM - 2:30 PM | ✅ PRIME TRADING HOURS | Best zone reactions |
| 2:30 - 3:00 PM | ⚠️ Only take A+ setups | Fading liquidity |
| 3:00 - 3:30 PM | ❌ NO NEW TRADES | Closing volatility |

### Market Condition Filters

| Condition | Action |
|-----------|--------|
| Nifty moving > 2% in one direction | Don't counter-trend trade |
| VIX > 20 (high volatility) | Widen SL slightly OR reduce position size |
| Major news day (RBI, Budget, Elections) | Don't trade or reduce size |
| Low volume day (holiday adjacent) | Avoid — zones may not hold |
| Gap up/down > 1% at open | Wait 30 min before trading |

### Zone-Specific Filters

| Filter | Rule | Reason |
|--------|------|--------|
| Zone too wide | Skip if zone height > 1.5% of price | SL too wide for intraday |
| Zone too thin | Skip if zone height < 0.1% of price | Too thin — noise zone |
| Too far from CMP | Skip if zone > 3% away from current price | Won't reach today |
| Overlapping zones | If supply and demand within 1%, SKIP | Confused area — no clear bias |
| Zone in dead area | Skip if between two major zones with no trend | Choppy, no follow-through |

### Stock Selection Filters

| Criteria | Rule |
|----------|------|
| **Liquidity** | Average daily volume > 5 lakh shares |
| **Spread** | Bid-ask spread < 0.05% |
| **Sector** | Not in the same sector as 2 other open positions |
| **News** | No earnings/results expected today |
| **Beta** | 0.5 - 2.0 (not too dull, not too wild) |

## 12. The Complete Trade Checklist

Before entering ANY trade, professionals check:

### Pre-Trade Checklist

```
□ 1. TREND: What is the higher-TF trend? (Am I trading WITH it?)
□ 2. ZONE: Is the zone clearly formed (leg-in → base → leg-out)?
□ 3. FRESH: Is the zone completely fresh (never tested)?
□ 4. QUALITY: Does it score ≥ 40/60 on the 6-dimension framework?
□ 5. WIDTH: Is zone width < 1.5% of price?
□ 6. DISTANCE: Is zone within 3% of current market price?
□ 7. TIME: Was zone formed recently (not ancient)?
□ 8. ARRIVAL: Is price approaching zone SLOWLY (not crashing)?
□ 9. VOLUME: Was the leg-out on high volume?
□ 10. RISK: Is SL < 1.5% of entry price?
□ 11. R:R: Is risk:reward at least 1:2?
□ 12. POSITION: Am I risking ≤ 1% of capital?
□ 13. EXPOSURE: Do I have < 5 open positions?
□ 14. DAILY LOSS: Am I within daily loss limit?
□ 15. TIME OF DAY: Is it within prime trading hours (9:45-2:30)?
□ 16. NO CONFLICT: No overlapping opposing zone nearby?
```

### Post-Trade Checklist

```
□ Trade entry logged (time, price, reason, zone score)
□ SL order placed immediately
□ Target order placed (or alerts set for partial)
□ Screenshot of zone saved for review
□ No more action until 1:1 R:R reached
```

### What's HARD to Automate

| Feature | Reason |

| Subjective "feel" for the market | Can't quantify experience/intuition |

| Sector rotation analysis | Too many variables for a zone bot |
| Pattern-based confirmation (complex) | Stick to simple candle rules |

### Feasibility Verdict

def score_zone_professional(zone, data, current_price):
    """
    Professional 6-dimension scoring.
    Each dimension: 0-10 points. Total: 60 max.
    Minimum to trade: 40/60
    """
    score = 0
    
    # Dimension 1: Departure Strength (0-10)
    if zone.has_gap or zone.leg_out_count >= 3:
        score += 10
    elif zone.leg_out_count >= 2 and zone.body_ratio > 0.7:
        score += 8
    elif zone.leg_out_body_pct >= 2 * avg_body:
        score += 6
    elif zone.leg_out_body_pct >= 1.5 * avg_body:
        score += 4
    else:
        score += 2
    
    # Dimension 2: Base Quality (0-10)
    if zone.base_candles == 1:
        score += 10
    elif zone.base_candles == 2:
        score += 8
    elif zone.base_candles == 3:
        score += 6
    elif zone.base_candles <= 5:
        score += 4
    else:
        score += 0  # Don't trade
    
    # Dimension 3: Freshness (0-10)
    if zone.is_fresh and zone.age_candles <= 50:
        score += 10
    elif zone.is_fresh and zone.age_candles <= 100:
        score += 7
    elif zone.is_fresh:
        score += 4
    else:
        score += 0  # Don't trade
    
    # Dimension 4: Arrival Quality (0-10)
    recent_candles = data.tail(5)
    avg_recent_body = recent_candles['body_pct'].mean()
    if avg_recent_body < 0.5 * avg_body:  # Small candles approaching
        score += 10
    elif avg_recent_body < avg_body:
        score += 7
    else:
        score += 3  # Momentum approach — riskier
